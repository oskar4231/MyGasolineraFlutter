Guía de Optimización de Imágenes
Aquí tienes los códigos que debes copiar y pegar para mejorar el rendimiento de las imágenes.

1. Archivo: 
lib/ajustes/facturas/controllers/crear_factura_controller.dart
Busca las funciones 
seleccionarImagenGaleria
 y 
tomarFoto
. Reemplázalas por estas versiones que incluyen maxWidth, maxHeight e imageQuality.

Future<XFile?> seleccionarImagenGaleria() async {
    return await _picker.pickImage(
      source: ImageSource.gallery,
      maxWidth: 1920,   // Redimensionar a Full HD máximo
      maxHeight: 1920,
      imageQuality: 80, // Compresión JPEG al 80%
    );
  }
  Future<XFile?> tomarFoto() async {
    return await _picker.pickImage(
      source: ImageSource.camera,
      maxWidth: 1920,
      maxHeight: 1920,
      imageQuality: 80,
    );
  }
